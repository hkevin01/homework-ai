# Homework AI

This project is designed to help create and organize solutions to various homework problems. Each homework assignment should be placed in its own subfolder within the project for modularity and clarity.

## Project Structure

- `src/` - Source code for solution generation and utilities.
- `assignments/` - Each homework assignment in its own subfolder.
- `tests/` - Unit and integration tests.
- `docs/` - Project documentation.
- `scripts/` - Helper scripts for automation and utilities.
- `.github/` - GitHub workflows and templates.
- `.copilot/` - Copilot configuration and prompts.

## Getting Started

1. Clone the repository.
2. Install dependencies (see below).
3. Add your homework assignments in the `assignments/` folder.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Add your homework assignment as a new folder under `assignments/` and place your solution code inside.
